# DamerBots

Site estatico da DamerBots.

## Como editar

O conteudo principal do site fica no arquivo `content.js`.

La voce pode alterar:

- textos da home
- posts do blog
- fotos
- equipe
- robos
- conquistas

## Como atualizar

1. Abra o arquivo `content.js`
2. Edite as informacoes
3. Salve
4. Atualize o site no navegador

## Arquivos principais

- `index.html`
- `style.css`
- `script.js`
- `content.js`

## Imagens

As imagens usadas no site ficam na mesma pasta do projeto.
